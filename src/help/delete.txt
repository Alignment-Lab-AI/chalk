%{H Usage: {appName} delete [COMMAND_OPTION]... [search path] }%
Deletes metadata from all artifacts specified in the search path.

%{T:<14:
Variable::Override::Type::Description
recursive::-r, -R, --recursive, --no-recursive::bool::Determines whether a file scan will recursively walk paths.
}%
Without configuration overrides, deleted objects will be 'published' to the 'extract' topic before deletion.

Note that there is currently no flag for specifying exceptions, so we recommend running this on specific artifacts.
